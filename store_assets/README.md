# store_assets（Web listing pack / template）

## App Name
HumanVIP Notepad (Test)

## Short Description
A lightweight note app with search, pin, share links, offline drafts, and voice dictation.

## Long Description
- Create and edit notes with autosave
- Search across titles and content
- Sort by updated/created/title
- Pin important notes on this device
- Share notes via read-only links
- Offline draft protection to avoid losing edits
- Voice dictation after creating a note (browser supported)

## Keywords
notes, notepad, voice dictation, share link, offline draft, autosave

## Screenshots checklist
- Notes list (search + sort + pin)
- Note editor (voice + autosave state)
- Share page (read-only)
- Offline draft restore prompt (optional)
