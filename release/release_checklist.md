# Release Checklist（Web）

## 1. 功能验收
- [ ] 按 spec/acceptance_tests.md 逐条通过
- [ ] 分享链接可用（/share/{id}）
- [ ] 搜索/排序/置顶稳定
- [ ] 离线草稿恢复正常
- [ ] 冲突提示可触发

## 2. 安全与合规（最低限）
- [ ] 配置生产环境 HTTPS
- [ ] 设置安全响应头（CSP/Frame/Content-Type 等，至少基本）
- [ ] 隐私政策可访问（/privacy 或静态页）
- [ ] 站点声明：本应用收集/不收集哪些数据

## 3. 运维
- [ ] 后端日志可查看
- [ ] 数据库备份策略（即使是测试也写明）
- [ ] 一键回滚方案（上一版本镜像/产物）

## 4. 构建与部署
- [ ] CI：lint/typecheck/build 通过
- [ ] 版本号与变更记录（可选）
- [ ] 域名解析正确：humanvip.com 指向生产环境
- [ ] 健康检查：/ 与 /api/health（如果有）可用
